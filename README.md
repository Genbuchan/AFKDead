# AFKDead

プレイヤーのAFK (Away From Keyboard)を検知し、指定の時間以降まで放置が続いたら自動的にサーバからKickするプラグインです。

## 導入方法

`afkdead-0.1.0.jar`をダウンロードし、お使いのMinecraftサーバの`plugins`フォルダに配置する。

サーバを起動、稼働中の場合は再起動を

## 設定項目 (config.yml)

```yaml
duration: 900s # seconds
```
